# Requisitos do Sistema

## Requisitos Funcionais

| ID   | Descrição |
|------|-----------|
| RF01 | O dashboard deve exibir a acurácia geral do modelo de detecção |
| RF02 | O dashboard deve exibir a matriz de confusão do modelo |
| RF03 | O dashboard deve mostrar a distribuição entre imagens reais e deepfakes no dataset |
| RF04 | O dashboard deve apresentar métricas de precisão, recall e F1-score |
| RF05 | O dashboard deve permitir filtrar os resultados por classe (real/fake) |
| RF06 | O dashboard deve exibir exemplos de imagens classificadas corretamente e incorretamente |
| RF07 | O dashboard deve apresentar a curva ROC e o valor de AUC |
| RF08 | O dashboard deve mostrar a evolução das métricas ao longo dos epochs de treinamento |

## Requisitos Não Funcionais

| ID    | Descrição |
|-------|-----------|
| RNF01 | O dashboard deve ser desenvolvido no Power BI |
| RNF02 | Os dados devem ser exportados do modelo em formato CSV ou Excel |
| RNF03 | O dashboard deve ser responsivo e legível em telas Full HD |
| RNF04 | A documentação deve ser escrita em Markdown no repositório GitHub |
| RNF05 | O repositório deve ser público no GitHub |
