# Diagrama de Caso de Uso

## Atores
- **Pesquisador/Analista**: usuário que acessa o dashboard para
  interpretar os resultados do modelo

## Casos de Uso

| ID    | Nome | Descrição |
|-------|------|-----------|
| UC01  | Visualizar métricas gerais | Ver acurácia, F1, precisão e recall do modelo |
| UC02  | Analisar matriz de confusão | Verificar distribuição de VP, FP, VN, FN |
| UC03  | Filtrar por classe | Filtrar visualizações por "Real" ou "Fake" |
| UC04  | Analisar curva ROC | Visualizar a curva ROC e o valor de AUC |
| UC05  | Acompanhar treinamento | Ver evolução das métricas por epoch |
| UC06  | Explorar distribuição do dataset | Ver proporção e volume de imagens por classe |
